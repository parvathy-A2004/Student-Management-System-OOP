# Student Management System (OOP)

## Description
A Python-based Student Management System developed using Object-Oriented Programming (OOP). The project allows users to manage student records through a menu-driven interface.

## Features
- Create Student
- View Students
- Search Student by ID
- Search Student by Name
- Search Student by Course
- Update Student
- Delete Student
- Count Students
- Count Students by Course
- Sort Students by ID
- Sort Students by Name
- Save Data to File
- Load Data from File

## Technologies Used
- Python 3
- OOP Concepts
- File Handling
