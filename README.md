Student Management System (OOP)

A Python-based Student Management System developed using Object-Oriented Programming (OOP) concepts. The project allows users to manage student records through a menu-driven interface.

-> Features
Create Student
View Students
Search Student by ID
Search Student by Name
Search Student by Course
Update Student Details
Delete Student
Count Total Students
Count Students by Course
Sort Students by ID
Sort Students by Name
Save Data to File
Load Data from File

-> Technologies Used
Python 3
Object-Oriented Programming (OOP)
File Handling
Files

student_oop.py – Main Python source code
stu_data.txt – Sample student data file
